## Paradigmas de Programación
Material de la materia **Paradigmas de Programación** (3º año – Lic. en Ciencias de la Computación, FaMAF).

- **Lab 1:** Programación funcional en **Haskell**  
- **Lab 2:** Implementación en **Java** de un lector automático de feeds como aplicación de consola  
- **Lab 3:** Extensión del lector de feeds para soportar **big data**, utilizando **Apache Spark** para el cálculo de entidades nombradas sobre grandes volúmenes de datos

Repositorio con fines académicos.
